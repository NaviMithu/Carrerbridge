# Career Connect App

Career Connect is a powerful platform designed to connect recruiters with talented job seekers. Our app streamlines the hiring process, making it easy for recruiters to find the right candidates and for job seekers to discover their dream jobs.

## Features

- **For Recruiters:**
  - Post and manage job listings
  - Search for and filter candidate profiles
  - View candidate resumes and profiles
  - Contact candidates directly through the platform

- **For Job Seekers:**
  - Create and update professional profiles
  - Browse and apply for jobs
  - Receive personalized job recommendations
  - Track application status

## Technologies Used
- Frontend: React 
- Backend: Node.js
- Database: PostgreSQL 
- Hosting: local host

## Getting Started

### Prerequisites
- Node.js version 14.x or above
- npm or yarn package manager
